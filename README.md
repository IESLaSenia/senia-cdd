# senia-firstboot

Senia firstboot is a simple tool to configure:
 
	* Hostname
	* IP
	* Netmask
	* Gateway
	* Server IP 

The Server IP is the current OCS Server and nagios server. At this moment this is the 
same server, but could be changed at near future.
